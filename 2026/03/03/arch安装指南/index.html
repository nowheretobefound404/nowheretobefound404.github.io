7.分区和格式化(使用Btrfs文件系统+luk2全盘加密)

7-1.分区

常用的分区配置:

  • / 根目录：>= 128GB（和用户主目录在同一个 Btrfs 文件系统上）
  • /home 用户主目录：>= 128GB（和根目录在同一个 Btrfs 文件系统上）
  • /boot EFI 分区：256MB（由电脑厂商或 Windows 决定，无需再次创建）
  • Swap 分区：>= 电脑实际运行内存的 60%（设置这个大小是为了配置休眠准备）

安装对应的包

安装 grub 到 efi分区:

接下来使用 vim 编辑 /etc/default/grub 文件:

进行如下修改:

  • 去掉 GRUB_CMDLINE_LINUX_DEFAULT 一行中最后的 quiet 参数
  • 把 loglevel 的数值从 3 改成 5。这样是为了后续如果出现系统错误，方便排错
  • 加入 nowatchdog 参数，这可以显著提高开关机速度
  • 去掉 GRUB_ENABLE_CRYPTDISK=y 前的注释，让grub启用对加密硬盘的支持
  • lsblk -f 查看加密前盘的UUID，在 GRUB_CMDLINE_LINUX_DEFAULT 中加入 rd.luks.name=你的未加密前盘的UUID=cryptroot root=/dev/mapper/cryptroot rootflags=subvol=@
  • 把 GRUB_SAVEDEFAULT 改成 true
  • 把 GRUB_DISABLE_SUBMENU 改成 y
  • 为了引导 win10，则还需要添加新的一行 GRUB_DISABLE_OS_PROBER=false
